# E-commerce Platform Search Function

This project implements and analyzes different search algorithms (Linear Search and Binary Search) to optimize product lookups on an e-commerce platform.

## Overview

Efficient searching is a core requirement for e-commerce systems to handle large product inventories. This application showcases:
* Linear Search for unsorted or small datasets.
* Binary Search for sorted datasets to achieve significantly faster lookups.
* A performance comparison regarding time complexity for both approaches.

---

## Project Structure

Based on the directory layout, the files are organized as follows:

* `Code/EcommerceSearch.java` - Contains the product data models, sorting mechanisms, and search algorithm implementations.
* `Output/` - Contains execution results and performance verification artifacts.
