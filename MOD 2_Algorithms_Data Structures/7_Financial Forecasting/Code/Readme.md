
# Financial Forecasting System

This project demonstrates financial forecasting utilizing a recursive approach to predict future business values based on constant growth rates.

## Overview

Financial forecasting models often rely on predicting cumulative growth over multiple periods. This implementation uses recursion in Java to calculate future financial metrics (such as revenue or asset growth) efficiently over a specified timeline, modeling complex growth patterns cleanly.

---

## Project Structure

Based on the directory layout, the files are organized as follows:

* `Code/FinancialForecast.java` - Contains the forecasting models and recursive algorithms calculating future economic values.
* `Output/Financial Forecasting_Output.jpg` - Image artifact verifying execution results, terminal logs, or generated growth trends.
